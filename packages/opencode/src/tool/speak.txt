Convert text to speech and play it aloud using a system or cloud TTS engine.

Use this tool when the user asks to:
- Read text aloud or "speak" something
- Listen to a response or message
- Use text-to-speech on any content

The tool supports multiple backends:
- System TTS: macOS `say`, Linux `espeak`/`espeak-ng`/`spd-say`, Windows PowerShell
- TypeWhisper: When the TypeWhisper HTTP API is running locally (http://localhost:8978)

Usage notes:
- The text parameter is required and contains the content to speak
- Voice and rate parameters are optional; sensible defaults are used
- On macOS, the system `say` command is used by default
- On Linux, the tool tries espeak-ng, espeak, then spd-say in order
- If TypeWhisper is available, it can be used as the TTS engine via its local API
- The tool runs asynchronously - speech plays in the background
